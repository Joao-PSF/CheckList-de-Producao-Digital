
# CheckList de Produção Digital

## Descrição
Sistema web para digitalização e gestão de checklists de produção, voltado para operadores e supervisores, com interface simples e responsiva, desenvolvido em PHP e MySQL.

## Requisitos Funcionais
- Login de operadores e supervisores
- Visualização de lista diária de tarefas por setor ou tipo de serviço
- Marcação de itens como "feito" com data e hora
- Campo para observações do operador
- Supervisores podem consultar os checklists preenchidos
- Geração automática de relatórios diários simples (em PDF ou tela)

## Requisitos Não-Funcionais
- Interface simples e com botões grandes, adequada ao uso em celulares
- Sistema web responsivo, acessível em rede local ou internet
- Baixa complexidade visual, com poucas telas e linguagem clara
- Banco de dados seguro com backup automático
- Tempo de carregamento abaixo de 2 segundos
- Desenvolvido em PHP e MySQL

## Critérios de Aceitação
- Redução de ao menos 80% dos checklists em papel
- Operadores conseguem usar o sistema sem necessidade de treinamento formal
- Supervisores acessam os registros em tempo real
- Sistema funcionando em celulares e computadores da produção
